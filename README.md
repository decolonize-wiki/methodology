# The decolonize.wiki methodology

This repository is the methodology behind [decolonize.wiki](https://decolonize.wiki):
a versioned, cited framework for identifying colonial framing in encyclopedic
text. Every flag the project publishes traces to a category in `categories/`;
every category traces to its sources.

## The equal-standing principle

This methodology draws on academic decolonial scholarship — Fanon, Césaire,
Said, Quijano, Mignolo, Dussel, Ngũgĩ wa Thiong'o, Rodney, Mbembe, Diop,
Mudimbe, Achebe, Memmi, Hountondji, Ndlovu-Gatsheni, Tamale, Oyěwùmí,
Tuhiwai Smith, Wynter, Glissant, C.L.R. James, Spivak, Guha, Lugones,
Rivera Cusicanqui, Coulthard, Leanne Betasamosake Simpson, and Brazil's
Black intellectual tradition (Abdias do Nascimento, Lélia Gonzalez, Beatriz
Nascimento, Sueli Carneiro, Conceição Evaristo), among others — **and** on
intellectual traditions that European-style academia has historically
excluded: Nêgo Bispo's contra-colonização, Ailton Krenak, Davi Kopenawa's
shamanic testimony, Steve Biko's movement writing, Amílcar Cabral's and
Thomas Sankara's speeches, Amadou Hampâté Bâ's defense of oral tradition,
quilombola and Indigenous oral traditions. Both carry equal standing here.
Treating only peer-reviewed writing as valid knowledge would reproduce the
problem this project critiques. This is itself a methodological commitment.
The full registry, with exact citations, is `SOURCES.md`.

## Structure

- `categories/` — one file per flag category (structured front-matter +
  criteria + exclusions + examples + citations). These files are read
  directly as analysis instructions; the methodology **is** the prompt.
- `GOVERNANCE.md` — how changes are proposed, judged, and versioned.
- `WIKIPEDIA-ENGAGEMENT.md` — how this project relates to Wikipedia and its
  editors.
- `SOURCES.md` — the curated source set that context facts must verify
  against.

## Versioning

Versions are git tags (`v0.1`, `v0.2`, …). Every published analysis records
the methodology version that produced it. v0.1 is deliberately small and
visibly incomplete — it improves in public.
